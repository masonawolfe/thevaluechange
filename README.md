# The Value Change

CPG strategy intelligence platform. Real-time signals, earnings analysis, and tactical execution plans for consumer goods professionals.

**Live site:** [thevaluechange.com](https://thevaluechange.com)

## Development

Single-file HTML app — all code lives in `index.html`. No build step required.

## Deployment

Push to `main` → Netlify auto-deploys to thevaluechange.com within ~30 seconds.

```bash
git add -A
git commit -m "your message"
git push
```
